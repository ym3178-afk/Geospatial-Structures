# Submission Checklist — The Last 800 Meters

## Before publishing

- [ ] Paste a complete Mapbox public token into `mapBox_Sketch_03.js`.
- [ ] Open the site with Live Server and confirm that restaurant points load.
- [ ] Test all four filters.
- [ ] Test Restaurant points / Density field.
- [ ] Click a restaurant and verify the popup.
- [ ] Test Campus view / District view.
- [ ] Test Export visible GeoJSON.
- [ ] Confirm that no token or loading panel covers the map.

## GitHub Pages

- [ ] Upload all files at the same repository level.
- [ ] Keep the exact filenames used by `index.html`.
- [ ] Enable GitHub Pages from the correct branch and folder.
- [ ] Open the public link in an incognito window.

## Screenshot composition

Use a desktop window around 1440 × 900:

1. choose `All cuisines`;
2. choose `0–800 m`;
3. choose `All records` and `All locations`;
4. use `Restaurant points`;
5. use `Campus view`;
6. open one representative restaurant popup;
7. keep the title, controls, legend, and research question visible;
8. make sure no browser console, token prompt, or error message appears.
